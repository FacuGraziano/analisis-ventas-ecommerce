# SQL

En esta carpeta se encuentran las consultas SQL utilizadas durante el desarrollo del proyecto de análisis de ventas ecommerce.

## 📂 Contenido

### 01_limpieza.sql
Incluye el proceso de limpieza y transformación de datos:
- detección de valores vacíos
- tratamiento de NULLs
- eliminación de devoluciones y cancelaciones
- exclusión de datos no comerciales
- construcción del dataset final listo para análisis

### 02_metricas.sql
Incluye las consultas utilizadas para el análisis exploratorio y generación de métricas:
- ingresos totales
- total de órdenes
- total de clientes
- ticket promedio
- top países
- top productos
- top clientes
- evolución mensual de ingresos
